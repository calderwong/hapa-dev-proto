# Design Philosophy for Mimi's Unified Content Hub

## Design Philosophy

**Color Palette**: Inspired by modern editorial design with a focus on mathematical elegance and technical precision. Using a sophisticated monochromatic base with strategic accent colors:
- Primary: Deep charcoal (#1a1a1a) and pure white (#ffffff)
- Accent: Warm amber (#f59e0b) for highlights and interactive elements
- Supporting: Cool gray (#6b7280) for secondary text and subtle elements

**Typography**: 
- Display: "Tiempos Headline" - A bold serif for headings that conveys academic authority
- Body: "Suisse Int'l" - A clean, highly readable sans-serif for content
- Code: "JetBrains Mono" - For technical content and code snippets

**Visual Language**: 
- Minimalist aesthetic with generous white space
- Grid-based layout reflecting mathematical precision
- Subtle geometric patterns and mathematical symbols as decorative elements
- Clean, technical documentation style inspired by research papers

## Visual Effects

**Used Libraries**:
- Anime.js for smooth micro-interactions and content reveals
- ECharts.js for data visualizations of project metrics
- p5.js for generative background patterns based on mathematical functions
- Splitting.js for sophisticated text animations
- Typed.js for dynamic typewriter effects in hero section

**Effects**:
- Subtle parallax scrolling for depth
- Typewriter animation for main heading
- Staggered fade-in animations for content sections
- Interactive hover states with smooth transitions
- Mathematical pattern generation in background using p5.js

**Animation Styling**:
- Easing: Cubic-bezier curves for natural, organic motion
- Duration: 200-400ms for micro-interactions, 800ms for content reveals
- Stagger: 50ms delays between elements for smooth sequential reveals

**Header Effect**:
- Clean navigation with subtle backdrop blur
- Smooth color transitions on scroll
- Mathematical symbol decorations that respond to scroll position

**Background**:
- Consistent mathematical pattern generated with p5.js
- Subtle geometric shapes based on linear algebra concepts
- Low opacity to maintain readability
- Responsive to user interactions
