# Feature Docs (Index)

This folder contains **big feature/workstream** documents.

Use this folder for:
- cross-cutting features that touch UI + pipelines + persistence
- multi-stage work where we need an explicit design + rollout plan
- features that need multi-agent coordination

If you’re starting a new workstream:
- create `docs/features/<FEATURE_NAME>.md`
- link it from `docs/README.md`
- if you will touch hot files, claim the work in `docs/.mind/Comms/NOTES_YYYY-MM-DD_<AGENT>.md`
